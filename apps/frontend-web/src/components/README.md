# Shared Presentation Components

1. **Style Consistency**: Rely exclusively on variables declared in [index.css](file:///c:/Users/ROG/Desktop/HIS/HIS2/apps/frontend-web/src/index.css). Do not inject ad-hoc styles.
2. **Stateless Definition**: Prefer functional components parsing props.
